# Mharc Bedielon Duller's Portfolio

This is my personal portfolio website, built with HTML and CSS. It highlights my experience as an IT Specialist and Data Analyst, showcasing my skills, education, and work history.

🔴🖤 The theme features a clean **red & black** design, supports **dark/light mode toggle**, and is fully **mobile responsive**.

## 🌐 Live Demo

Visit the live site here: [https://therealmharc.github.io](https://therealmharc.github.io)

## 💼 Features

- 🎨 Minimal red/black color palette
- 🌓 Manual dark/light mode toggle (saved via localStorage)
- 📱 Mobile-first, fully responsive layout
- 👤 Collapsible sections for About, Skills, Work, and Education
- 🖼️ GitHub avatar used as favicon and header profile image
- 🧠 Clean semantic HTML with accessibility in mind
